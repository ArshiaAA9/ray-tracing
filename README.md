# Ray-Tracing 

A learning orientated project focusing on simulating realtime raytracing with objects and object shadows.  
Features interactive light movement and dynamic shadow casting.  

Built with C++, OpenGL, and SFML. Features interactive light movement and dynamic shadow casting.

# Build
Cloning and building the repo:
```
git clone https://github.com/ArshiaAA9/ray-tracing
cd ray-tracing
cmake -B build
cmake --build build
```
Running the simulation:
```
cd build
./sudotu
```
